Purpose :-
Run Away Game Based On Blueprint Endless Runner Shooter Template for Unreal Engine that provides core functionalities like modular tile creation, obstacles and power-ups spawning and a mechanism to change gameplay style during runtime. EndlessRunnerShooterGameMode :-
Choose Endless RunnerShooterGameMode to be Game Mode in your level. This will set Default Pawn Class to EndlessRunnerShooterPawn automatically. GameManager :- Add the GameManager blueprint to your level. This blueprint controls the game state (Running, Shooting, Flying, etc..) and also determines which game state to choose depending on the current one.
