Air Tap Game Demo

Air Tap is a side scrolling game, like the popular game Super Mario Brothers, where the player controls a bird in a 2D environment. Unlike Mario, the player only has control over the bird's vertical movement. The main goal is for the bird to travel as far as it can without hitting any dark purple pipes. I created this game using Unity Game Engine and bunch of C# codes like... Bird Script Pipe script Pipes Spawner Script Basic Controllers and Collider and used UI for the Score and Game Over screen
